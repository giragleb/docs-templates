## Package contents

In the box:

2x Reach RTK module

2x Tallysman TW4721 antenna with adhesive mount

2x DF13 <-> DF13 straight cable

2x DF13 <-> jumper wires cable

2x Micro-USB <-> USB cable

2x Micro-USB <-> USB-F OTG cable
